Detailed Description

Author
Jonas Lukasczyk (jl@jl.nosp@m.uk.d.nosp@m.e)
Date
01.09.2019

This filter creates an icosphere with a specified radius, center, and number of subdivisions at each vertex of an input vtkPointSet.

See also
ttk::IcoSphere
ttk::ttkAlgorithm
