Detailed Description

TTK VTK-filter that dilates or erodes a specified vertex label.

See also
ttk::MorphologicalOperators
Author
Jonas Lukasczyk (jl@jl.nosp@m.uk.d.nosp@m.e)
Date
01.02.2019

VTK wrapping code for the @MorphologicalOperators package.

This filter either a) dilates a specified label by assigning the label of a corresponding vertex to all its neighbors, or b) erodes a specified label by assigning to a corresponding vertex the largest label among its neighbors.

The input data array that will be dilated or eroded needs to be specified via the standard VTK call SetInputArrayToProcess() with the following parameters:

Parameters
idx0 (FIXED: the first array the algorithm requires)
port0 (FIXED: first port)
connection0 (FIXED: first connection)
fieldAssociation0 (FIXED: point data)
arrayName(DYNAMIC: string identifier of the VTK array)
