Detailed Description

TTK VTK-filter that computes the global identifiers for each vertex and each cell as point data and cell data scalar fields.

Author
Julien Tierny julie.nosp@m.n.ti.nosp@m.erny@.nosp@m.lip6.nosp@m..fr
Date
August 2015.

This filter is useful to retrieve the global identifiers of vertices or cells in subsequent filters throughout the VTK pipeline.

Parameters
InputInput data-set (vtkDataSet)
OutputOutput data-set with identifier fields (vtkDataSet)

This filter can be used as any other VTK filter (for instance, by using the sequence of calls SetInputData(), Update(), GetOutput()).

See the related ParaView example state files for usage examples within a VTK pipeline.
