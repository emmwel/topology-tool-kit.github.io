Detailed Description

TTK VTK-filter that wraps the triangulation processing package.

Author
Guillaume Favelier guill.nosp@m.aume.nosp@m..fave.nosp@m.lier.nosp@m.@lip6.nosp@m..fr
Date
April 2017.

VTK wrapping code for the @Triangulation package.

Parameters
InputGeometry, either 2D or 3D, either regular grid or triangulation (vtkDataSet)
OutputOutput geometry requested by the user (vtkDataSet)

This filter can be used as any other VTK filter (for instance, by using the sequence of calls SetInputData(), Update(), GetOutput()).

See the related ParaView example state files for usage examples within a VTK pipeline.

See also
ttk::Triangulation
