Detailed Description

TTK abstractMorseSmaleComplex processing package.

Parent class containing convenience functions shared between Morse-Smale Complex algorithms for 2D and 3D domains.

Author
Guillaume Favelier guill.nosp@m.aume.nosp@m..fave.nosp@m.lier.nosp@m.@lip6.nosp@m..fr
Julien Tierny julie.nosp@m.n.ti.nosp@m.erny@.nosp@m.lip6.nosp@m..fr
Date
February 2017.

AbstractMorseSmaleComplex is a processing package managing the inputs, outputs and dependencies of the MorseSmaleComplex2D and MorseSmaleComplex3D packages.

See also
ttk::Triangulation
