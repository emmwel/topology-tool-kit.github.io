Detailed Description

The MergeTreeDistanceMatrix class provides methods to compute the distance between multiple merge trees and output a distance matrix.

Author
Mathieu Pont mathi.nosp@m.eu.p.nosp@m.ont@l.nosp@m.ip6..nosp@m.fr
Date
2021.

This VTK filter uses the ttk::MergeTreeDistanceMatrix module to compute the distance matrix of a group of merge trees.
