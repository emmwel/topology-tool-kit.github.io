Detailed Description

TTK VTK-filter that works in conjunction with the ttkEndFor filter to iterate over blocks, rows, array values, and arrays.

Author
Jonas Lukasczyk jl@jl.nosp@m.uk.d.nosp@m.e
Date
01.01.2019

This filter works in conjunction with the ttkEndFor filter to iterate over blocks, rows, array values, and arrays.

Parameters
InputvktObject either a vtkMultiBlockDataSet, vtkTable, or vtkDataSet
OutputvktObject one element of the input
