Detailed Description

TTK VTK-filter that arranges vtkDataSets on a grid.

Author
Jonas Lukasczyk jl@jl.nosp@m.uk.d.nosp@m.e
Date
01.10.2018

This filter arranges vtkDataSets which are stored in a vtkMultiBlockDataSet on a grid.
