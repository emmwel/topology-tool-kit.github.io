Detailed Description

Performs color mapping of a scalar field.

Author
Jonas Lukasczyk jl@jl.nosp@m.uk.d.nosp@m.e
Date
01.11.2020
Parameters
InputvtkImageData.
OutputvtkImageData.

This class maps each value of a scalar point data array to either a solid color, a predefined color map, or a manually defined color map.

Related Publication: "Cinema Database Specification - Dietrich Release v1.2". D. Rogers, J. Woodring, J. Ahrens, J. Patchett, and J. Lukasczyk. Technical Report LA-UR-17-25072, Los Alamos National Laboratory, 2018.
