◆ setupTriangulation()

int ttk::blank::Blank::setupTriangulation ( AbstractTriangulation *  triangulation)
inline

Setup a (valid) triangulation object for this TTK base object.

Precondition
This function should be called prior to any usage of this TTK object, in a clearly distinct pre-processing step that involves no traversal or computation at all. An error will be returned otherwise.
Note
It is recommended to exclude this pre-processing function from any time performance measurement. Therefore, it is recommended to call this function ONLY in the pre-processing steps of your program. Note however, that your triangulation object must be valid when calling this function (i.e. you should have filled it at this point, see the setInput*() functions of ttk::Triangulation). See ttkBlank for further examples.
Parameters
triangulationPointer to a valid triangulation.
Returns
Returns 0 upon success, negative values otherwise.
See also
ttk::Triangulation
