Detailed Description

TTK VTK-filter for surface quadrangulation.

Author
Pierre Guillou pierr.nosp@m.e.gu.nosp@m.illou.nosp@m.@lip.nosp@m.6.fr
Date
March 2019

The current filter transforms a triangulated surface into a quadrangulated one.

Parameters
Input0Input triangular surface (2D) geometry (vtkDataSet)
OutputQuadrangular mesh (vtkDataSet)

This filter can be used as any other VTK filter (for instance, by using the sequence of calls SetInputData(), Update(), GetOutput()).

See the related ParaView example state files for usage examples within a VTK pipeline.

This method specifies in the port information the data type of the output objects. It is possible to either explicitly specify a type by adding a vtkDataObject::DATA_TYPE_NAME() key, or to pass a type of an input port to an output port by adding the ttkAlgorithm::SAME_DATA_TYPE_AS_INPUT_PORT() key.

This method has to be overridden to specify the data types of the outputs.
